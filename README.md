# 交易所批量工具（Python GUI）

## 1. 当前版本功能
- 顶部菜单：`币安提现`、`bg交易所1对多`、`链上`
- 右上角全局显示公网 IP（支持手动检测）
- 表格支持勾选、全选、滚动条、状态流转（等待中/进行中/完成/失败）
- 默认模拟执行（`dry run`）

### 币安提现
- 同页支持 `多对多` / `1对多` 模式切换
- 支持固定数量 / 随机数（2位小数）/ 全部
- 支持按余额刷新币种、查询余额（全币种）、按币种加载网络
- 查询/刷新时优先使用勾选项；未勾选时会自动使用全部可用数据

### Bitget 1对多
- 地址导入后自动全选
- 支持固定数量 / 随机数（2位小数）/ 全部
- 支持查询余额（全币种）

### 链上（EVM）
- 同页支持 `多对多` / `1对多` / `多对1`
- 网络支持：`ETH`、`BSC`
- 币种支持：原生币 + 预置 `USDT/USDC`，并支持“合约搜索并选择”自定义 ERC20
- 数量支持：固定 / 随机（2位小数）/ 全部
- 支持执行间隔、执行线程数
- 三种模式都支持查询余额（优先勾选，未勾选则自动查全部可用数据）
- 导入后自动全选

## 2. 启动
- 双击 `启动.command`
- 或终端执行 `python3 main.py`

## 3. 链上导入格式
### 多对多
每行：`私钥或助记词 接收地址`（接收地址放最后一列）

### 1对多
上方填写 1 个转出钱包私钥/助记词，列表每行导入 1 个接收地址

### 多对1
上方填写 1 个收款地址，列表每行导入 1 个私钥/助记词

## 4. 依赖说明（链上）
链上真实签名依赖 `eth-account`：

```bash
pip install eth-account
```

如果未安装，链上转账/助记词解析会给出明确提示。

## 5. 一键回归
执行：

```bash
bash scripts/selfcheck.sh
```

脚本会执行语法检查与冒烟测试。

## 7. 凭证加密存储
- API Key / API Secret / Passphrase / 链上私钥或助记词会加密后写入 `data/*.json`。
- 默认本地密钥文件：`data/.secret.key`（首次保存时自动生成）。
- 也可通过环境变量 `WITHDRAW_SECRET_KEY` 指定 32 字节 Base64 密钥。
- 注意：如果丢失密钥文件且未保留环境变量密钥，历史加密配置将无法解密。
