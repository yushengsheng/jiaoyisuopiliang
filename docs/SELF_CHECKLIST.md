# 本地手工回归清单

## 1. 启动与界面
- 启动 `python3 main.py` 无报错。
- 窗口缩放时，配置区与表格不重叠、不遮挡。
- 菜单页可切换：`币安提现`、`bg交易所1对多`、`链上`。

## 2. 多对多模式
- 切换到 `多对多`。
- 粘贴导入 `api_key api_secret address` 三列数据。
- 导入后账号自动全选。
- 勾选列可单独反选，`全选/取消全选` 正常。
- `删除选中` 正常删除。

## 3. 一对多模式
- 切换到 `1对多`。
- 填写源账号 `API Key/API Secret`。
- 粘贴导入地址（每行一个地址）。
- 导入后地址自动全选。
- 表格显示地址列表，状态列初始为 `-`。

## 4. 参数校验
- 不设置网络执行提现，弹出 `未设置提现网络，无法提现`。
- 数量模式 `固定数量`：输入非数字或 <=0 应报错。
- 数量模式 `随机数`：最小值/最大值格式错误应报错。
- 数量模式 `全部`：1对多模式下多地址执行应拦截并提示。

## 5. 余额/币种
- `查询余额（全币种）` 后，币安 / Bitget 的余额列更新，币种下拉选项同步更新。
- 未勾选时，提示先勾选至少一个账号；1对多模式下至少需要可用源账号。
- 网络下拉应随币种变化刷新。

## 6. 提现执行状态
- 点击批量提现后，状态流转为：
  - 等待中的行：`等待中`
  - 正在执行的行：`进行中`
  - 完成后：`完成` 或 `失败`
- 线程数为 2 时，同时处于 `进行中` 的行不应长期超过 2。

## 7. 配置持久化
- 点击 `保存配置` 后退出重启。
- 模式、币种、网络、数量模式、线程、间隔可恢复。
